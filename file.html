Steps for uploading NFT and creating it's json file

1. Go to the pinata.cloud (link below) and create your account, it's free

2. After login, go to dashboard and upload your artwork

3. After sucessful upload, you will get a CID hash, something like "QmVnVzv6GudA1US1BGawha97Yv5t4UPH7cnT1ZoSvz8tES"

4. Download the json file (link below) and change the attribute value according to your artwork

5. Very important! in json file, under "image" property you have to provide the link of your artwork

6. The link should be like this "https://ipfs.io/ipfs/your_artwork_CID_hash_here"

7. Now save the json file and upload the same and get the CID hash for json file

8. Prefix the https://ipfs.io/ipfs/your_json_file_CID_hash_here

9. Now you will get link or your NFT json file, something like "https://ipfs.io/ipfs/QmVnVzv6GudA1US1BGawha97Yv5t4UPH7cnT1ZoSvz8tES"

10. Use this link to mint your NFT.
